paloaltonetworks.panos.panos_api_key module – retrieve api_key for username/password combination

Note

This module is part of the paloaltonetworks.panos collection (version 2.10.2).

To install it, use: ansible-galaxy collection install paloaltonetworks.panos. You need further requirements to be able to use this module, see Requirements for details.

To use it in a playbook, specify: paloaltonetworks.panos.panos_api_key.

New in paloaltonetworks.panos 1.0.0

  • Requirements

  • Parameters

  • Notes

  • Examples

  • Return Values

Synopsis

  • This module will allow retrieval of the api_key for a given username/password

Requirements

The below requirements are needed on the host that executes this module.

  • pan-python can be obtained from PyPI https://pypi.python.org/pypi/pan-python

  • pandevice can be obtained from PyPI https://pypi.python.org/pypi/pandevice

Notes

Note

  • Panorama is supported.

  • Checkmode is NOT supported.

  • PAN-OS connectivity should be specified using provider or the classic PAN-OS connectivity params (ip_address, username, password, api_key, and port). If both are present, then the classic params are ignored.

Examples

- name: retrieve api_key
  panos_api_key:
    provider: '{{ provider }}'
  register: auth

- debug:
    msg: '{{ auth.api_key }}'
