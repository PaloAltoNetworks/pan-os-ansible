panos_bgp_policy_rule -- Configures a BGP Policy Import/Export Rule
===================================================================

.. contents::
   :local:
   :depth: 1


Synopsis
--------

Use BGP to publish and consume routes from disparate networks.




Requirements
------------
The below requirements are needed on the host that executes this module.

- pan-python can be obtained from PyPI https://pypi.python.org/pypi/pan-python
- pandevice can be obtained from PyPI https://pypi.python.org/pypi/pandevice



Parameters
----------

  type (required, type: str)
    The type of rule.


  name (required, type: str)
    Name of filter.


  enable (type: bool, default: True)
    Enable rule.

    Address Family Identifier.


  match_safi (type: str)

  match_nexthop (type: list)
    Next-hop attributes.

    AS-path regular expression.


  match_community_regex (type: str)
    Community AS-path regular expression.


  match_extended_community_regex (type: str)
    Extended Community AS-path regular expression.


  used_by (type: list)
    Peer-groups that use this rule.


  action (type: str)
    Rule action.


  action_local_preference (type: int)
    New local preference value.


  action_med (type: int)
    New MED value.


  action_nexthop (type: str)
    Nexthop address.


  action_origin (type: str)
    New route origin.


  action_as_path_limit (type: int)
    Add AS path limit attribute if it does not exist.


  action_as_path_type (type: str)
    AS path update options.


  action_as_path_prepend_times (type: int)
    Prepend local AS for specified number of times.


  action_community_type (type: str)
    Community update options.


  action_community_argument (type: str)
    Argument to the action community value if needed.


  action_extended_community_type (type: str)
    Extended community update options.


  action_extended_community_argument (type: str)
    Argument to the action extended community value if needed.


  action_dampening (type: str)
    Route flap dampening profile; only with "import" type.


  action_weight (type: int)
    New weight value; only with "import" type.


  address_prefix (type: list)
    List of address prefix strings or dicts with "name"/"exact" keys.

    If a list entry is a string, then *exact=False* for that name.


  vr_name (type: str, default: default)
    Name of the virtual router; it must already exist; see :ref:`panos_virtual_router <panos_virtual_router_module>`.


  commit (type: bool)
    Commit configuration if changed.


  provider (type: )
    A dict object containing connection details.


    ip_address (type: str)
      The IP address or hostname of the PAN-OS device being configured.


    username (type: str, default: admin)
      The username to use for authentication.  This is ignored if *api_key* is specified.


    password (type: str)
      The password to use for authentication.  This is ignored if *api_key* is specified.


    api_key (type: str)
      The API key to use instead of generating it using *username* / *password*.


    port (type: int, default: 443)
      The port number to connect to the PAN-OS device on.


    serial_number (type: str)
      The serial number of a firewall to use for targeted commands. If *ip_address* is not a Panorama PAN-OS device, then this param is ignored.

Notes
-----

.. note::
   - Checkmode is supported.
   - Panorama is supported.
   - PAN-OS connectivity should be specified using *provider* or the classic PAN-OS connectivity params (*ip_address*, *username*, *password*, *api_key*, and *port*).  If both are present, then the classic params are ignored.
   - If the PAN-OS to be configured is Panorama, either *template* or *template_stack* must be specified.




Examples
--------

.. code-block:: yaml+jinja

    
    # Add a BGP Policy
      - name: Create Policy Import Rule
        panos_bgp_policy_rule:
          provider: '{{ provider }}'
          vr_name: 'default'
          name: 'import-rule-001'
          type: 'import'
          enable: true
          action: 'allow'
          address_prefix:
            - '10.1.1.0/24'
            - name: '10.1.2.0/24'
              exact: false
            - name: '10.1.3.0/24'
              exact: true
          action_dampening: 'dampening-profile'

      - name: Create Policy Export Rule
        panos_bgp_policy_rule:
          provider: '{{ provider }}'
          vr_name: 'default'
          name: 'export-rule-001'
          type: 'export'
          enable: true
          action: 'allow'

      - name: Remove Export Rule
        panos_bgp_policy_rule:
          provider: '{{ provider }}'
          state: 'absent'
          vr_name: 'default'
          name: 'export-rule-001'
          type: 'export'
