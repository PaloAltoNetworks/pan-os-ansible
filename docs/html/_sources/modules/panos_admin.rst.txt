.. _panos_admin_module:


panos_admin -- Add or modify PAN-OS user accounts password.
===========================================================

.. contents::
   :local:
   :depth: 1


Synopsis
--------

PanOS module that allows changes to the user account passwords by doing API calls to the Firewall using pan-api as the protocol.


.. WARNING::
   This module is **deprecated**, and will be removed in version 3.0.0
   of the collection.

   This module is a subset of M(panos_administrator)'s functionality.

   Use M(panos_administrator) instead.



Requirements
------------
The below requirements are needed on the host that executes this module.

- pan-python can be obtained from PyPI https://pypi.python.org/pypi/pan-python



Parameters
----------

  ip_address (required, type: str)
    IP address (or hostname) of PAN-OS device being configured.


  port (type: str, default: 443)
    Port used to connect to the PAN-OS device being configured.


  username (type: str, default: admin)
    Username credentials to use for auth unless *api_key* is set.


  password (type: str)
    Password credentials to use for auth unless *api_key* is set.


  api_key (type: str)
    API key that can be used instead of *username*/*password* credentials.


  admin_username (type: str, default: admin)
    Username that needs password change.


  admin_password (required, type: str)
    New password for *admin_username* user


  role (type: str)
    role for admin user


  commit (type: bool)
    Commit configuration if changed.





Notes
-----

.. note::
   - Checkmode is not supported.




Examples
--------

.. code-block:: yaml+jinja

    
    # Set the password of user admin to "badpassword"
      - name: set admin password
        panos_admin:
          ip_address: "192.168.1.1"
          password: "admin"
          admin_username: admin
          admin_password: "badpassword"



Return Values
-------------

status (success, str, okey dokey)
  success status
