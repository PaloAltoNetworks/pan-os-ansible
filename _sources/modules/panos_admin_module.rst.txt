.. Title

paloaltonetworks.panos.panos_admin module -- Add or modify PAN-OS user accounts password.
+++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++

.. Collection note

.. note::
    This module is part of the `paloaltonetworks.panos collection <https://galaxy.ansible.com/paloaltonetworks/panos>`_ (version 2.10.2).

    To install it, use: :code:`ansible-galaxy collection install paloaltonetworks.panos`.
    You need further requirements to be able to use this module,
    see :ref:`Requirements <ansible_collections.paloaltonetworks.panos.panos_admin_module_requirements>` for details.

    To use it in a playbook, specify: :code:`paloaltonetworks.panos.panos_admin`.

.. Deprecated

DEPRECATED
----------
:Removed in: version 3.0.0
:Why: This module is a subset of \ :strong:`ERROR while parsing` :literal:`M('panos\_administrator')`\ : :strong:`parameter 'panos\_administrator' is not a FQCN`\ 's functionality.
:Alternative: Use \ :strong:`ERROR while parsing` :literal:`M('panos\_administrator')`\ : :strong:`parameter 'panos\_administrator' is not a FQCN`\  instead.

Synopsis
--------

.. Description

- PanOS module that allows changes to the user account passwords by doing API calls to the Firewall using pan-api as the protocol.

------------
The below requirements are needed on the host that executes this module.

- pan-python can be obtained from PyPI \ https://pypi.python.org/pypi/pan-python\

.. Examples

Examples
--------

.. code-block:: yaml+jinja

    
    # Set the password of user admin to "badpassword"
      - name: set admin password
        panos_admin:
          ip_address: "192.168.1.1"
          password: "admin"
          admin_username: admin
          admin_password: "badpassword"
