.. note::
    This module is part of the `paloaltonetworks.panos collection <https://galaxy.ansible.com/paloaltonetworks/panos>`_ (version 2.19.1).

    see :ref:`Requirements <ansible_collections.paloaltonetworks.panos.panos_http_server_module_requirements>` for details.

Requirements

The below requirements are needed on the host that executes this module.

- pan-python
- pandevice \>= 0.11.1
- PAN-OS \>= 8.0

.. Notes

Notes
-----

.. note::
   - Panorama is supported.
   - Check mode is supported.
   - PAN-OS connectivity should be specified using \ :emphasis:`provider`\  or the classic PAN-OS connectivity params (\ :emphasis:`ip\_address`\ , \ :emphasis:`username`\ , \ :emphasis:`password`\ , \ :emphasis:`api\_key`\ , and \ :emphasis:`port`\ ).  If both are present, then the classic params are ignored.
