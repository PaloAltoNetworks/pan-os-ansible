.. Title

paloaltonetworks.panos.panos_sag module -- Create a static address group.
+++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++

.. Collection note

.. note::
    This module is part of the `paloaltonetworks.panos collection <https://galaxy.ansible.com/paloaltonetworks/panos>`_ (version 2.13.1).

    To install it, use: :code:`ansible-galaxy collection install paloaltonetworks.panos`.
    You need further requirements to be able to use this module,
    see :ref:`Requirements <ansible_collections.paloaltonetworks.panos.panos_sag_module_requirements>` for details.

    To use it in a playbook, specify: :code:`paloaltonetworks.panos.panos_sag`.

Synopsis
--------

.. Description

- Create a static address group object in the firewall used for policy rules.

Requirements
------------
The below requirements are needed on the host that executes this module.

- pan-python can be obtained from PyPI \ https://pypi.python.org/pypi/pan-python\ 
- pandevice can be obtained from PyPI \ https://pypi.python.org/pypi/pandevice\ 
- xmltodict can be obtained from PyPI \ https://pypi.python.org/pypi/xmltodict\

Examples

        tags: ["tags to be associated with the group", ]
