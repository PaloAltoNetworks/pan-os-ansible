.. Collection note

.. note::
    This httpapi plugin is part of the `paloaltonetworks.panos collection <https://galaxy.ansible.com/ui/repo/published/paloaltonetworks/panos/>`_ (version 3.2.0).

    It is not included in ``ansible-core``.
    To check whether it is installed, run :code:`ansible-galaxy collection list`.

    To install it, use: :code:`ansible\-galaxy collection install paloaltonetworks.panos`.

    To use it in a playbook, specify: :code:`paloaltonetworks.panos.panos`.
